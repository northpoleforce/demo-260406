## ✨ 核心特性

1. **实时同步** - WebSocket + 心跳 + 自动重连
2. **色彩编码** - 黄蓝绿灰红，直观显示订单状态
3. **催单提醒** - 声音+视觉双重提醒，可静音
4. **离线处理** - 断线重连，数据同步
5. **触控优化** - 48px按钮，反馈动画
6. **配置灵活** - 二次确认、声音、排序均可配置
7. **统计实时** - 30秒自动刷新，手动刷新支持

## 📝 注意事项

1. **环境配置**: 部署前需配置 `.env` 文件，设置正确的后端地址
2. **浏览器要求**: Chrome 90+, Safari 14+, Firefox 88+
3. **网络要求**: 需与边缘主控在同一局域网
4. **设备要求**: 10-12英寸触控平板，最低2GB内存
